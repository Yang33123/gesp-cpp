Lesson 2：cout 输出进阶

🎯 学习目标

  • 理解 << 的作用
  • 掌握多次输出
  • 理解换行符

📘 知识讲解

cout 用于向屏幕输出内容。

cout << "Hello"; cout << "World";

输出会连在一起：HelloWorld

换行方法

cout << "Hello" << endl; cout << "World";

endl 表示换行。

💻 示例代码

#include <iostream> using namespace std; int main() { cout << "I am learning C++" << endl; cout << "This is Level 1"; return 0; }

⚠ 常见错误

  • 忘记加分号 ;
  • 写成 single <
