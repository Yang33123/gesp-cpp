Lesson 1：第一个 C++ 程序

🎯 学习目标

  • 理解 C++ 程序基本结构
  • 学会使用 cout 输出

📌 标准程序结构

#include <iostream> using namespace std; int main() { cout << "Hello World"; return 0; }

🧠 结构解析

#include <iostream>：引入输入输出库

using namespace std;：使用标准命名空间

int main()：程序入口

cout：输出语句

⚠ 常见错误

  • 忘记分号 ;
  • 拼错 cout
  • 引号不成对

✅ 本课小练习

输出三行文字：

  • Hello
  • My Name is Tom
  • I Love C++
